# Hand Gesture Control System

A real-time hand gesture recognition system built using Python, OpenCV, and MediaPipe that enables touchless control of mouse and media functions.

## Features

- Real-time hand tracking using MediaPipe
- Mouse control using finger tracking
- Media control (play, pause, next, previous)
- Gesture smoothing and cooldown for stability
- Modular and extensible architecture

## Technologies Used

- Python
- OpenCV
- MediaPipe
- NumPy
- PyAutoGUI
